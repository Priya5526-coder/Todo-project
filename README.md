# Django To-Do App
A simple To-Do application built with Django.  
Users can add, update, and delete tasks with a clean and responsive interface.

## Features
- Create, Read, Update, Delete (CRUD) tasks
- Mark tasks as complete/incomplete
- User authentication (login/signup)
- Responsive UI with HTML, CSS, and Bootstrap

## Tech Stack
- Python
- Django
- HTML, CSS, Bootstrap
- SQLite (default database)
